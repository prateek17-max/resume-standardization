# Resume Standardizer

A minimal Python project to extract, process, and standardize resumes (PDF/DOCX) using customizable templates and LLM-powered extraction.
The project uses Streamlit for the UI/frontend, providing a simple and interactive user experience.

## Features
- Extracts text from PDF and DOCX resumes
- Uses LLM for information extraction
- Generates standardized resumes using templates
- Simple file handling and formatting modules

## Project Structure
- `app.py` — Main application entry point
- `modules/` — Core logic (file handling, extraction, formatting, generation)
- `templates/` — Resume templates (DOCX)
- `tests/` — Unit tests

## Setup
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Place your resume files in the project directory.

## Usage
Run the main application:
```bash
python app.py
```

## Customization
- Edit or add templates in the `templates/` folder.
- Adjust extraction/formatting logic in `modules/` as needed.
